# Structure files of the repo

- **assets** - Includes: `js`, `css`, and `img` files.

- **/assets/img** - Contains folder `page` for containing some misc stuff, like UI eyecandy or for just some popup in the future, `staff` are the profile faces of the staff, and `students` contain student's faces

- **node_modules** - since the whole thing is powered with node, this folder contains all the plugin dependency of this project, thanks to npm for saving it here

- **testdatabase** - an sqlite3 file, somehow with no extension attached by DBeaver
